# 📦 ZimCarry  
### AI 기반 여행 짐 기내·위탁 여부 검사 Android 앱

---

## ✈️ 프로젝트 소개

**ZimCarry**는 여행 전 짐을 준비할 때  
기내 반입이 가능한지, 위탁 수하물로 보내야 하는지,  
혹은 반입이 금지된 물품인지 **AI로 빠르게 안내해주는 Android 애플리케이션**입니다.

사용자는 물품을 **카메라로 촬영**하기만 하면  
AI가 물품을 인식하고,  
항공 보안 규정을 기반으로 **3단계 결과(기내 / 위탁 / 금지)**를 제공합니다.

> 여행 전 헷갈리는 짐 규정을  
> **촬영 한 번으로 10초 안에 확인**하는 것을 목표로 합니다.

---

## 🎯 기획 배경

여행을 준비할 때 가장 자주 겪는 불편함 중 하나는  
**“이 물건, 기내에 들고 타도 되나?”**라는 의문입니다.

- 보조배터리, 액체류, 면도기, 칼 등  
  규정이 자주 바뀌고 헷갈리는 물품이 많음
- 인터넷 검색은 정보가 제각각이라 신뢰하기 어려움
- 공항 보안 검색대에서 처음 문제를 알게 되는 경우도 많음

ZimCarry는 이러한 문제를 해결하기 위해  
**AI 인식 + 규정 기반 판단**을 결합한 앱으로 기획되었습니다.

---

## 🤖 주요 기능

### 1️⃣ AI 물품 자동 인식
- 카메라로 물품을 촬영하면 AI가 물품 종류를 인식
- 별도의 입력 없이 즉시 분류 결과 제공

### 2️⃣ 기내 · 위탁 · 금지 여부 안내
- 항공 보안 규정을 기준으로 3단계 결과 제공
  - ✅ 기내 반입 가능
  - 📦 위탁 수하물
  - ❌ 반입 금지
- 결과와 함께 간단한 설명 제공

### 3️⃣ 빠르고 간편한 검사 흐름
- 복잡한 설정 없이 **촬영 → 결과 확인** 단순한 UX
- 초보 사용자도 바로 사용할 수 있도록 설계
